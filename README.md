# CoffeeStats :coffee::snake:
Tools to help make projections in [BizCafe](https://www.interpretive.com/business-simulations/bizcafe/)

## Requirements
- `python` >= 3.6

## Data
`data.xlsx` holds weekly sales information:

| Week | Cups Sold | Hours Open |
|------|-----------|------------|
| 1    | 1050      | 80         |
| 2    | 2040      | 80         |
| 3    | 3000      | 100        |

**do not to change the cell structure - just add rows in the current format**

## Running
```
python CoffeeStats/CoffeeStats.py
```
